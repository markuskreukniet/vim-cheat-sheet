# vim-cheat-sheet

| Command    | Mode         | Description                                |
| ---------- | ------------ | ------------------------------------------ |
| h, j, k, l | Normal       | Move the cursor left, down, up, and right. |
| Esc        | Any          | Return to Normal mode.                     |
| :w         | Command-line | Save the current file.                     |
| :q         | Command-line | Quit Vim.                                  |
